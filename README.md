# Student Expense Tracker using AWS

This is a simple serverless AWS project that generates a monthly
expense report and hosts it as a static website.

## Technologies Used
- AWS S3 (Static Website Hosting)
- AWS Lambda (Python)
- AWS IAM

## Project Description
A Lambda function calculates student expenses and generates
an HTML page. The page is uploaded to S3 and served publicly.

## Live Demo
https://student-expense-sanjana.s3-website.ap-south-1.amazonaws.com/
